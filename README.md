# 💵 분철이지 (Buncheol-Easy)

> 가격 고민도, DM 부담도 없이. 익명 금액 제시로 시작하는 분철

_분철이지는 K-pop 팬덤의 독특한 공동구매 문화인 '분철'을 누구나 쉽고 안전하게 경험할 수 있도록 돕는 서비스입니다._

<br>

## ❓ '분철'이란?

K-pop, 애니메이션 등 다양한 팬덤에서 굿즈를 공동구매하여 멤버(캐릭터)별로 나누는 행위를 '분철'이라고 합니다.

팬들이 번거로운 과정을 감수하며 분철을 하는 이유는 단 하나, 내 최애의 아이템을 확정적으로 갖기 위해서입니다.

### 굿즈의 특수성

팬덤 굿즈의 상당수는 랜덤성을 띕니다.

- 특전 (포토카드): 멤버 랜덤
- 인형/키링: 멤버 또는 색상 랜덤
- 앨범/CD: 버전 랜덤

멤버가 10명인 그룹이라면? → 내가 원하는 멤버를 얻을 확률은 고작 1/10

#### 하지만!

멤버 수만큼 세트를 구매하면 → 판매처가 중복 없이 모든 멤버/버전을 보내주는 시스템이 존재합니다.

팬들은 이 시스템을 활용해 각자 다른 멤버를 원하는 사람들을 모아 1세트를 함께 구매한 뒤, 각자의 최애 아이템을 나누어 가집니다.

<br>

### 기존 분철의 문제점

현재 분철은 주로 X(구 트위터)를 통해 수동으로 이루어지고 있으며, 개최자와 참여자 모두에게 부담이 큽니다.

#### 👤 개최자의 고민

| 문제            | 상세                                                  |
|---------------|-----------------------------------------------------|
| 💸 가격 선정의 부담  | 인기 멤버에 프리미엄을 얼마나 붙여야 할까? 너무 높으면 신청자가 없고, 너무 낮으면 손해  |
| 💬 관리의 번거로움   | 신청자마다 일일이 DM으로 입금 확인, 주소 받기, 송장 전달, 수령 확인           |
| 📦 배송 추적의 어려움 | 누가 받았고 누가 안 받았는지 일일이 체크하고 물어봐야 함                    |

#### 🙋 참여자의 고민

| 문제 | 상세 |
|------|------|
| 😰 DM 보내기 부담 | 처음 보는 사람에게 DM을 먼저 보내는 게 부담스러움 |
| 💭 가격 제안의 두려움 | "이 가격 괜찮을까요?"라고 물어보는 게 너무 조심스러움. 너무 낮으면? 너무 높으면? |
| 🔍 분철 찾기 어려움 | 타임라인에 뒤섞인 수많은 게시물 속에서 원하는 멤버가 남은 분철 찾기 힘듦 |
| ⏰ 선착순의 압박 | "이 멤버 아직 남았나요?" 조급하게 DM 보내야 함 |

<br>

분철이지는 이렇게 부담스럽고 번거로운 과정을 자동화하여, 분철이 모든 팬들에게 쉽고 당연한 문화로 자리잡길 바라며 시작한 프로젝트입니다!

<br>

## 👍 분철이지는 아래와 같은 기능을 제공합니다

분철이지는 익명 금액 제시 시스템과 자동화된 관리 기능으로 이 모든 불편함을 해소합니다.

### 🎪 개최자를 위한 기능

- 익명 금액 제시 시스템
    - 즉시구매가만 설정하거나
    - 최소가를 설정하고 금액 제시를 받거나
    - 멤버별로 다르게 설정 가능
- 인기 멤버는 자연스럽게 높은 제시가가, 비인기 멤버는 낮은 가격이 형성됩니다
- 에스크로 기반 자동 결제 시스템
- 참여자가 직접 배송지 등록
- 간편한 운송장 번호 등록
- 운송장 자동 조회 시스템
- 수령 확인 시 자동 정산 처리

### 🙋 참여자를 위한 기능

- 익명으로 부담 없이 원하는 금액 제시
- 내 제시 금액이 몇 등인지 실시간 순위 확인
- 개최자와 DM 필요 없이, 모든 과정이 플랫폼 안에서 자동으로
- 원하는 그룹/멤버로 바로 분철 검색
- 어떤 멤버가 남았는지 한눈에 확인
- 클릭 한 번으로 즉시구매 or 금액 제시
- 비인기 멤버는 낮은 가격에 제시하여 합리적으로 얻을 수 있어요
- 인기 멤버는 내가 낼 수 있는 최대 가격까지 제시 가능
